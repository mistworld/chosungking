# 배포 및 바인딩 확인 단계

## 1단계: 배포 완료 확인
배포가 완료되었는지 확인하세요.

**확인 방법:**
- 배포 명령어 실행 후 "Successfully deployed" 또는 "Total Upload" 메시지가 나왔는지
- 에러 메시지가 있었는지

## 2단계: Dashboard에서 바인딩 확인
Cloudflare Dashboard에서 확인:

1. https://dash.cloudflare.com 접속
2. Workers & Pages → `steep-moon-7816` 클릭
3. Settings → Variables (또는 Bindings) 클릭
4. 확인할 항목:
   - `ROOM_LIST` (KV Namespace) - 있어야 함
   - `WORD_CACHE_NEW` (KV Namespace) - **이게 있는지 확인!**
   - `GAME_STATE` (Durable Object) - 있어야 함

## 3단계: WORD_CACHE_NEW가 없으면?
Dashboard에서 수동으로 추가:

1. Settings → Variables → Add binding
2. Type: KV Namespace 선택
3. Variable name: `WORD_CACHE_NEW` 입력
4. KV namespace: 드롭다운에서 선택 (ID: 2c001bb3f40c41df8e79974421a3268c)
5. Save

## 4단계: 테스트
브라우저 콘솔에서:
```javascript
fetch('https://steep-moon-7816.sucksuck1114.workers.dev/api/validate-word', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ word: '가가' })
}).then(r => r.json()).then(data => {
  console.log('source:', data.source);
  console.log('_bindingName:', data._bindingName);
});
```
