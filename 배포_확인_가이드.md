# 배포 확인 가이드

## 방법 1: 강제 재배포

CMD에서 다음 명령어 실행:

```
wrangler deploy --config wrangler-do.toml --force
```

## 방법 2: 배포 로그 자세히 보기

배포할 때 나오는 모든 메시지를 복사해서 확인하세요.
특히 다음 메시지가 있는지 확인:
- "Applied migration"
- "Uploaded"
- "Published"
- 에러 메시지가 있는지

## 방법 3: 계정 확인

1. CMD에서 다음 명령어 실행:
   ```
   wrangler whoami
   ```
2. 나오는 이메일 주소 확인
3. Dashboard에서 같은 계정으로 로그인했는지 확인

## 방법 4: Workers 목록 확인

1. Dashboard → Workers & Pages → Overview
2. `chosung-game-do` Worker가 있는지 확인
3. 있으면 클릭해서 상세 화면 확인

## 방법 5: 다른 위치에서 확인

1. Dashboard → Workers & Pages
2. 왼쪽 메뉴에서 **Workers** 클릭
3. `chosung-game-do` 찾기
4. 클릭 → **Settings** 탭
5. **Durable Objects** 섹션 확인
