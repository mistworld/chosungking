# 최종 단계 - Pages에서 바인딩

## ✅ 완료된 작업
- Worker 배포 성공 (`chosung-game-state-do`)
- Durable Object 바인딩 생성됨 (`GameStateRoom`)

## 🎯 다음 단계: Pages에서 바인딩

### 1단계: Dashboard 열기
1. https://dash.cloudflare.com 접속
2. 로그인

### 2단계: Pages 프로젝트로 이동
1. 왼쪽 메뉴에서 **Workers & Pages** 클릭
2. **Pages** 탭 클릭
3. `chosung-scramble` 프로젝트 클릭

### 3단계: Settings 열기
1. 상단 메뉴에서 **Settings** 클릭
2. 왼쪽 메뉴에서 **Functions** 클릭

### 4단계: Durable Object 바인딩 추가
1. **Durable Object Bindings** 섹션 찾기
2. **Add binding** 버튼 클릭
3. 다음 정보 입력:
   - **Variable name**: `GAME_STATE`
   - **Durable Object**: 드롭다운 클릭 → `chosung-game-state-do` 선택
   - **Class name**: `GameStateRoom`
4. **Save** 클릭

### 5단계: 확인
- 바인딩이 추가되면 목록에 `GAME_STATE`가 보입니다
- 이제 Pages에서 Durable Objects를 사용할 수 있습니다!

## ⚠️ 중요 사항

### "No option"이 나오면?
- Dashboard에서 **Workers & Pages** → **Overview**로 가서
- `chosung-game-state-do` Worker가 있는지 확인
- 없다면 몇 분 더 기다린 후 다시 시도

### Worker가 Dashboard에 안 보이면?
1. 5-10분 기다리기 (Cloudflare가 처리하는데 시간이 걸릴 수 있음)
2. 페이지 새로고침 (F5)
3. 다른 브라우저로 시도

## 🎉 성공 확인

바인딩이 추가되면:
- ✅ Pages에서 `context.env.GAME_STATE`를 사용할 수 있습니다
- ✅ `functions/api/game-state.js`가 정상 작동합니다
- ✅ 점수 계산이 실시간으로 동기화됩니다!
