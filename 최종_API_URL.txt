# 최종 API URL

## Cloudflare Workers API 주소
https://chosung-game-v2.sucksuck1114.workers.dev

## API 엔드포인트

### 1. 방 목록 조회
GET https://chosung-game-v2.sucksuck1114.workers.dev/api/rooms

### 2. 방 생성
POST https://chosung-game-v2.sucksuck1114.workers.dev/api/create-room

### 3. 게임 상태 조회
GET https://chosung-game-v2.sucksuck1114.workers.dev/api/game-state?roomId={roomId}

### 4. 방 참가
POST https://chosung-game-v2.sucksuck1114.workers.dev/api/join-room

### 5. 단어 검증
POST https://chosung-game-v2.sucksuck1114.workers.dev/api/validate-word

### 6. 채팅
GET/POST https://chosung-game-v2.sucksuck1114.workers.dev/api/chat?roomId={roomId}

## Vercel index.html에서 변경할 내용

모든 `/api/`를 다음으로 변경:
`https://chosung-game-v2.sucksuck1114.workers.dev/api/`

예:
- `fetch('/api/rooms')` → `fetch('https://chosung-game-v2.sucksuck1114.workers.dev/api/rooms')`
