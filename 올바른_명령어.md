# 올바른 Wrangler 명령어

## ❌ 잘못된 명령어
```cmd
wrangler list
```
→ Wrangler 4.49.0에서는 이 명령어가 없습니다

## ✅ 올바른 명령어

### 배포된 Worker 확인
```cmd
wrangler deployments --config wrangler-do.toml
```

또는 특정 Worker의 배포 목록 확인:
```cmd
wrangler deployments chosung-game-state-do
```

### Dashboard에서 확인
1. Cloudflare Dashboard → **Workers & Pages** → **Overview**
2. 여기서 모든 Worker 목록을 볼 수 있습니다

## "Assertion failed" 에러

이 에러는 Wrangler의 내부 경고일 수 있습니다.
배포 자체는 성공했을 수도 있으니 Dashboard에서 확인해보세요.
