# 문제 해결 방법

## 현재 상황
- `chosung-game-do` Worker가 Dashboard에 안 보임
- `gamestateroom` Worker는 보임
- Pages에서 바인딩할 때 "No option"만 나옴

## 해결 방법

### 방법 1: Worker 이름 확인 및 재배포

1. CMD에서 다음 명령어로 배포된 Worker 목록 확인:
   ```cmd
   wrangler list
   ```

2. `chosung-game-do`가 목록에 있는지 확인

3. 없다면 다시 배포:
   ```cmd
   wrangler deploy --config wrangler-do.toml
   ```

### 방법 2: 기존 `gamestateroom` Worker 사용

`gamestateroom` Worker가 이미 있다면, 그 Worker에 DO를 추가하는 방법도 있습니다.

### 방법 3: 완전히 새로운 이름으로 배포

Worker 이름을 완전히 새로운 것으로 변경해서 배포해보세요.
