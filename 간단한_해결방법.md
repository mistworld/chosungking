# Durable Objects 등록 문제 해결 (초보자용)

## 문제
- `wrangler deploy`는 성공하지만 Dashboard에 Durable Objects가 안 보임
- "No Durable Objects found" 메시지

## 해결 방법 (단계별)

### 1단계: 마이그레이션 태그 변경

마이그레이션이 이미 적용되었다고 인식되어 새로 등록이 안 될 수 있습니다.
태그를 변경해서 다시 배포해보세요.

### 2단계: 배포 명령어 실행

CMD 창에서 다음 명령어를 실행하세요:

```
wrangler deploy --config wrangler-do.toml
```

### 3단계: 배포 로그 확인

배포할 때 다음 메시지가 나와야 합니다:
- "Applied migration v1-game-state"
- "Uploaded chosung-game-do"
- "Published chosung-game-do"

### 4단계: Dashboard 확인

배포 후 1-2분 기다린 다음:
1. Cloudflare Dashboard 열기
2. 왼쪽 메뉴에서 **Workers & Pages** 클릭
3. **Durable Objects** 탭 클릭
4. `GameStateRoom` 또는 `chosung-game-do`가 보이는지 확인
